header

object

오디오 뿐만 아니라 비디오, 자바 애플릿, PDF 등 다양한 멀티미디어 파일을 삽입할 때 사용

웹문서 안에 다른 문서를 삽입할 때도 사용

PDF 삽입 예제

embed

초기부터 사용해서 대부분의 브라우저에서 사용 가능

src 속성사용, width, height, 닫는 태그 없음

사용가능 종류

비디오: mp4, webm

오디오: mp3, mp4, m4a

예제

<embed src="음악파일.mp3" type="">

audio, video

html5 웹표준 제정이후 가장 많이 변한 부분

그 전까지는 플러그인 프로그램이 따로 필요했으나, html5 부터는 브라우저 안에서 삽입 후 바로 재생 가능

단, 브라우저마다 플레이어가 다르다는 건 함정 -_-;;

속성

controls: 사용자가 컨트롤 할 수 있도록 함

autoplay: 오디오나 비디오를 자동으로 재생

muted: 음소거

자동 재생을 하려면 muted(음소거) 속성을 추가해야 함

loop: 반복 재생

preload: 페이지를 불러올 때, 파일을 어떻게 로딩할 지 지정. 가능한 값은 auto, metadata, none. 기본값은 preload="auto"

width, height: 너비, 높이. 하나만 지정할 경우 나머지는 자동으로 계산 적용

poster="파일이름": video 태그에서 사용. 재생 전까지 화면에 표시될 포스터 이미지를 지정

오디오 파일을 배경음악처럼 사용하려면, autoplay loop 속성 지정. 플레이어를 숨기려면 controls 를 빼면 됨. 단, 대부분의 브라우저에서 자동 재생 금지함. -_-;;

예제

muted 속성을 추가해야 autoplay 가능

<audio src="파일 경로"></audio>

<video src="파일 경로" controls></video>

<video src="파일 경로" controls width="너비값"></video>
