반응형 웹과 미디어 쿼리

뷰포트

스마트폰 화면에서 실제로 내용이 표시되는 내용

뷰포트 지정하기

  • width 뷰포트 너비
  • height 뷰포트 높이
  • user-scalable 확대 | 축소 가능 여부 (yes | no)
  • initial-scale 확대 | 축소 초기값 (기본값: 1)

<meta name="viewport" content="width=device-width, initial-scale=1.0">

뷰포트 단위

vw(viewport width): 1vw 는 뷰포트 너비의 1% 와 같다
vh(viewport height): 1vh 는 뷰포트 높이의 1% 와 같다
vmin(viewport minimum): 뷰포트의 너비와 높이 중 작은 값의 1% 와 같다
vmax(viewport maximum): 뷰포트의 너비와 높이 중 큰 값의 1% 와 같다
vmin, vmax 테스트

미디어 쿼리

@media [only | not] 미디어유형 and (조건) * and (조건)

  only: 미디어 쿼리 미지원 브라우저에서는 무시하고 실행하지 않음
  not: not 다음에 지정하는 미디어 유형 제외 (예: not tv - TV 유형은 제외)
  and: 여러 개의 조건 추가 가능 (예: and (max-width: 768px))

미디어 유형

어디에서 사용할 것인지 지정함

  • all 모든 유형
  • print 인쇄 장치
  • screen 컴퓨터 및 스마트폰의 스크린
  • tv 음성과 영상이 동시에 출력되는 TV
  • aural 음성합성장치(주로 화면을 읽어 소리로 출력해 주는 장치)
  • braille 점자 표시 장치
  • handheld 패드처럼 손에 들고 다니는 장치
  • projection 프로젝터
  • tty 디스플레이 기능이 제한된 장치, 이런 장치에서는 px 단위 사용 불가
  • embossed 점자 프린터

예: (min-width: 400px) && (max-width: 450px)

해당하지 않음

단말기의 가로 너비와 세로 높이 속성

  • device-width, device-height 가로 너비, 세로 높이 지정
  • min-device-width, min-device-height 최소 너비, 최소 높이 지정
  • max-device-width, max-device-height 최대 너비, 최대 높이 지정

아이폰 X 에 적용하려면

@media screen and (min-device-width: 375px) and (min-device-height: 812px) {
  .....
}

화면 회전 속성

orientation 속성을 사용하면, 스마트 폰이나 태플릿에서 화면을 가로나 세로로 돌릴 때, 기기의 방향을 확인할 수 있다

  • orientation: portrait 세로 모드
  • orientation: landscape 가로 모드

아이폰 X의 가로 모드는?

@media screen and (min-device-width: 812px) and (orientation: landscape) {
  .....
}

이 외에도 화면 비율, 단말기 화면 비율, 색상당 비트 수 같은 여러가지 미디어 쿼리 조건이 있다 (w3c에서 확인)

미디어 쿼리의 중단점

  • 화면 크기에 따라 다른 css 를 적용할 분기점을 중단점이라고 한다.
  • 모든 기기를 반영할 수는 없으므로, 모바일 | 태블릿 | 데스크톱 정도로만 구분
  • 모바일 퍼스트 기법: 모바일이 제약 조건이 많으므로, 모바일을 기본으로 작성 (태플릿과 PC의 기본으로도 적용됨)
    • 스마트폰 미디어 쿼리 없이 기본 CSS로 작성
      방향까지 고려한다면, min-width 의 세로를 portrait 320px, landscape 480px 로 지정
    • 태블릿 세로 크기가 768px 이상일 경우, 가로 크기는 데스크톱과 같이 1024px 이상으로 지정
    • 데스크탑 화면 크기가 1024px 이상일 경우 지정

외부 CSS 파일 연결하기

  • @import 문으로 CSS 내부에서 사용
  • <link rel="stylesheet" media="screen" href="파일주소"> 로 head 태그 안에서 사용 (주로 사용됨)

항목을 배치하는 속성

justify-content: 주축 방향의 정렬 방법
align-items: 교차축 방향의 정렬 방법
align-self: 교차축에 있는 개별 항목의 정렬 방법
align-content: 교차축에서 여러 줄로 표시된 항목의 정렬 방법

플렉스 컨테이너 지정하는 display 속성

display: flex (컨테이너 안의 플렉스 항목을 블록 레벨 요소로 배치);
display: inline-flex (인라인 레벨 요소로 배치);

flex-direction 플렉스 방향을 지정

row: (주축을 가로로 지정 및 왼쪽에서 오른쪽으로 배치. 기본값)
row-reverse: (주축을 가로로 지정 및 오른쪽에서 왼쪽으로 배치)
column: (주축을 세로로 지정 및 위쪽에서 아래쪽으로 배치)
column-reverse: (주축을 세로로 지정 및 아래쪽에서 위쪽으로 배치)

플렉스 항목의 줄을 바꾸는 flex-wrap 속성

nowrap: 줄바꿈 없음, 한 줄에 표시, 기본값
wrap: 줄바꿈
wrap-reverse: 줄을 바꾸고, 시작점과 끝점을 바꿈. 즉 줄이 반대로 바뀜(아래에서 위 또는 오른쪽에서 왼쪽으로)

배치 방향과 줄 바꿈을 한번에 지정: flex-flow

row wrap: 왼쪽에서 오른쪽, 줄바꿈
row nowrap: 왼쪽에서 오른쪽, 한 줄로

주축 정렬 방법 지정: justify-content

flex-start: 주축의 시작점에 맞춰 배치
flex-end: 주축의 끝점에 맞춰 배치
center: 주축의 중앙에 맞춰 배치
space-between: 첫 번째 항목과 끝 항목을 주축의 시작점과 끝점에 배치한 후 나머지 중간 항목들은 같은 간격으로 배치
space-around: 모든 항목을 주축에 동일한 간격으로 배치

교차축 정렬 방법 지정: align-items

flex-start: 교차축의 시작점에 맞춰 배치
flex-end: 교차축의 끝점에 맞춰 배치
center: 교차축의 중앙에 배치
baseline: 교차축의 문자 기준선에 맞춰 배치
stretch: 플렉스 항목을 늘려 교차축에 가득 차게 배치

여러 줄일 때 교차축 정렬 방법을 지정하는 속성

flex-start: 교차축의 시작점에 맞춰 배치
flex-end: 교차축의 끝점에 맞춰 배치
center: 교차축의 중앙에 맞춰 배치
space-between: 처음과 끝 항목은 시작과 끝에 맞추고 중간 항목들은 동일한 간격으로 배치
space-around: 모든 항목을 동일한 간격으로 배치
stretch: 항목을 늘려서 교차축에 가득 차게 배치

CSS 그리드 레이아웃

플렉스 레이아웃은 1차원이지만, CSS 그리드 레이아웃은 2차원

수평, 수직 어느 방향이든 배치 가능

레고 블록을 맞추듯 할 수 있음

대부분의 모던 브라우저에서 사용가능

특징

가로 방향의 "줄"과 세로 방향의 "칼럼"으로 화면 구성

가장 최신의 레이아웃 제작 방법

그리드 컨테이너를 지정하는 속성

grid: 컨테이너 안의 항목을 블록 레벨 요소로 배치
inline-grid: 인라인 레벨 요소로 배치

칼럼과 줄을 제어하는 속성들

너비 200px 칼럼 * 3개
grid-template-columns: 200px 200px 200px


줄 높이 100px
grid-template-rows: 100px


상대크기 1대 1대 1인 칼럼 * 3개
grid-template-columns: 1fr 1fr 1fr


상대크기 1대 1대 1인 칼럼 * 3개를 반복함수로
grid-template-columns: repeat(3, 1fr)


minmax 함수
줄 높이 최소 100px, 최대 auto
grid-template-rows: minmax(100px, auto)


자동으로 칼럼 갯수를 조절하는 값
너비 200px인 칼럼을 화면 너비에 가득 찰 때까지 배치
grid-template-columns: repeat(auto-fit, 200px)


auto-fit: 화면이 넓을 때에는 남는 공간 없이 채움
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))


auto-fill: 칼럼의 최소 너비만 표시하고 남는 공간을 채우지 않음
grid-template-columns: repeat(auto-fill, minmax(200px, 1fr))


그리드 항목을 지정하는 속성
column-gap: 컬럼과 컬럼사이의 간격 지정 (20px)
row-gap: 줄과 줄 사이의 간격 지정 (30px)
gap: 두 종류의 간격을 한번에 지정 (20px 30px)
플렉스의 justify-content, align-content 등의 속성도 사용 가능 


그리드 라인을 이용해 배치하는 속성
grid-column-start: 칼럼 시작의 라인 번호 지정
grid-column-end: 칼럼 마지막 라인 번호 지정
grid-column: 시작, 마지막 번호 한번에 (예: 1/4)
grid-row-start: 줄 시작의 라인 번호 지정
grid-row-end: 줄 마지막 라인 번호 지정
grid-row: 시작, 마지막 번호 한번에 (예: 2/4)


템플릿 영역을 만들어 배치
번호를 사용하지 않아도 되고, 직관적이라 편리
빈 곳은 .(마침표) 로 표시
.box-1 { grid-area: box1 }
.box-2 { grid-area: box2 }

grid-template-areas:
  "box1  box1  box2"
  "box1  box1  box2"
  "  .    .    box2"
