CSS와 박스 모델

크롬 개발자모드에서 요소(elememt)의 계산됨(computed) 항목에서 해당 박스가 차지하는 모든 크기를 확인할 수 있다

박스 주요 스타일 정리

box-sizing:
  border-box - 테두리까지 포함해서 너비값 지정(주로 사용)
  content-box - 콘텐츠 영역만 너비값 지정(기본값)

box-shadow:
  수평 수직 흐림정도 번짐정도 색상 inset(그림자를 안쪽으로 처리)

border-style:
  none - 테두리 없음, 기본값
  hidden - 숨김
  solid - 실선으로
  dotted - 점선으로
  dashed - 짧은 직선으로
  double - 테두리를 이중선으로
  groove - 테두리를 창에 조각한 것처럼 표시,
           홈이 파인듯 한 입체감
  inset - outset 과 반대, groove 와 똑같이 표시
  outset - 표에서 seperate 일 경우 전체 박스테두리가 창에서
    튀어나온 것처럼 표시, collapse 일 경우 ridge 와 똑같이 표시
  ridge - 테두리를 창에서 튀어나온 것처럼 표시

border-width:
  크기 - 임의 크기(대부분)
  thin - 작게
  medium - 중간
  thick - 두껍게

border-color: crimson 모두 통일 혹은 아래처럼 각각 지정 가능
border-top-color: coral
border-right-color: rgb(204, 247, 14)
border-bottom-color: rgb(29, 240, 53)
border-left-color: rgb(39, 36, 226)

border-radius: 둥글게 처리, 공백으로 4부분 다르게 적용가능
border-radius: 가로 반지름 / 세로 반지름 (타원을 만들경우)

테스트

border-radius: 7rem(가로반지름) / 3rem(세로반지름)

특정 꼭지점만 적용할 경우

border-top-left-radius: 3rem(가로반지름) 5rem(세로반지름)

margin

(주의) 세로로 정렬시 마진이 겹치면 큰 마진 값을 기준으로 합쳐진다

배치

img {
  float: left;
  margin: .5rem 2rem 1rem 0;
  width: 200px;
  border-radius: .5rem;
}

'깐부 할아버지'로 잘 알려진 배우 오영수 씨가 드라마 '오징어 게임' 속 연기로 한국 배우 최초로 미국 골든글로브 연기상을 수상했습니다.

오 씨가 받은 상은 골든글로브 TV 부문 남우조연상입니다.

TV 드라마 남우주연상은 '석세션'의 제메리스트롱에게 돌아가, 이 부문 후보에 오른 이정재 씨의 수상은 불발됐습니다.

'오징어 게임'은 456억 원의 상금을 차지하기 위해 목숨을 걸고 벌이는 서바이벌 게임을 그린 드라마로, 오 씨는 '깐부 할아버지'라는 별칭을 얻은 게임 참가자 오일남 역을 맡았습니다.

clear 속성 (float 를 해제함)

clear:
  left - float: left 해제
  right - float: right 해제
  both - float 모두 해제

위치 지정하기

position:
  static - 문서 흐름에 따라 배치 (기본값)
  relative - 위치값 지정이 가능함 (그외 static 과 동일)
  absolute - relative 값을 지정한 상위 요소를 기준으로 배치
  fixed - 브라우저 창을 기준으로 배치

배경색과 배경범위 지정

background-color

background-color 속성은 상속되지 않는다. 기본적으로 모든 요소의 배경색이 투명이므로 body의 색을 지정하면 투과되어 보이는 것 뿐이다.

background-clip

background-clip: 적용범위를 지정
  border-box - 박스 모델의 가장 외관인 테두리까지 적용(기본값)
  padding-box - 테두리를 뺀 패딩 범위까지 적용
  content-box - 내용(콘텐츠) 부분에만 적용

background-position: 백그라운드 이미지의 위치를 지정

background-position: 수평위치 수직위치
  수평위치 - left | center | right | 백분율 | 길이값
  수직위치 - top | center | bottom | 백분율 | 길이값

백분율로 표시할 경우 모서리로 부터 떨어진 위치에 이미지의 떨어진 위치를 지정하는 것이다.

30% 70% 일 경우, 요소의 가로-30% 세로-70% 인 위치에, 이미지의 가로-30% 세로-70% 의 부분을 맞추는 것

의도와는 다른 결과가 나올 수 있다. -_-;;

background-origin: 배경 이미지의 적용 범위 조절

background-clip 과 비슷해 보임

background-origin:
  content-box - 내용 부분에만 배경 이미지 표시 (기본값)
  padding-box - 패딩까지 표시
  border-box - 테두리까지 표시

background-size:
  auto - 원래 이미지 크기 (기본값)
  contain - 요소 안에 배경이미지가 모두 들어오도록
            이미지를 확대, 축소
  cover - 요소를 모두 덮도록 확대, 축소
  크기 - 너비와 높이를 임의 지정, 하나만 지정할 경우
        나머지는 자동으로 계산
  백분율 - 요소의 크기를 기준으로 백분율 적용

선택자

div ~ p { 형제 선택자: div 다음에 오는 모든 p 요소 }
a[href] { 속성 선택자: href 속성이 있는 a 요소 }
a[target = _blank] {
  target 이 _blank 인 a 요소
  이 요소에만 새창 아이콘을 추가하려는 경우 등
}
[class ~= mybtn] { 포함 요소 선택: class 명 중에 mybtn 이
        포함되어 있다면 선택}
a[title |= us] { title 속성값에 us 또는 us-... 처럼 -으로
        연결된 것이 있다면 선택}
a[title ^= eng] { title 속성값이 eng 로 시작하는 a 요소 선택:
        eng, english 등}
[href $= xls] { href 속성값이 xls 로 끝나는 요소 선택:
        확장자 사용시 유용}
[href *= w3] { href 속성값에 w3 이 어느 위치든 포함되어
        있다면 선택}

구조 가상 클래스
:only-child { 부모 안에 자식 요소가 하나뿐일 때, 자식 요소 선택}
A:only-type-of { 부모 안에 A 요소가 하나뿐일 때 선택}
:first-child { 부모 안의 모든 요소중에서 첫 번째 자식 요소 선택}
:last-child {부모 안의 모든 요소중에서 마지막 자식 요소 선택}
A:first-of-type { A 요소 중, 첫 요소의 타입 지정}
A:last-of-type { A 요소 중, 마지막 요소의 타입 지정}
:nth-child(n) { n 번째 요소}
:nth-last-child(n) { 끝에서 n 번째 요소}
A:nth-of-type(n) { A 요소 중, n 번째 요소}
A:nth-last-of-type(n) { A 요소 중, 끝에서 n 번째 요소}

::after { 지정한 요소 내부의 내용 뒤에 추가}
