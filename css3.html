트랜지션과 애니메이션

transform 과 변형 함수

transform: translate(50px[x축], 100px[y축]);

2차원 변형함수 종류

transform:
  translate(tx, ty) { 지정한 크기만큼 x, y 로 이동 }
  translateX(tx) { 지정한 크기만큼 x 로 이동 }
  translateY(ty) { 지정한 크기만큼 y 로 이동}
  scale(sx, sy) { 지정한 크기만큼 x, y 로 확대|축소 }
  scaleX(sx) { 지정한 크기만큼 x 로 확대|축소 }
  scaleY(sy) { 지정한 크기만큼 y 로 확대|축소 }
  rotate(각도) { 지정한 각도만큼 회전 }
  skew(ax, ay) { 지정한 각도만큼 x, y 로 왜곡 }
  skewX(ax) { 지정한 각도만큼 x 로 왜곡 }
  skewY(ay) { 지정한 각도만큼 y 로 왜곡 }

3차원 변형함수 종류

transform:
  translate3d(tx, ty, tx) { 지정한 크기만큼 x, y, z 로 이동 }
  translateZ(tz) { 지정한 크기만큼 z 로 이동 }
  scale3d(sx, sy, sz) { 지정한 크기만큼 x, y, x 로 확대|축소 }
  scaleZ(sz) { 지정한 크기만큼 z 로 확대|축소 }
  rotate(rx, ry, 각도) { 지정한 각도만큼 회전 }
  rotate3d(rx, ry, rz, 각도) { 지정한 각도만큼 회전 }
  rotateX(각도) { 지정한 각도만큼 x 로 회전 }
  rotateY(각도) { 지정한 각도만큼 y 로 회전 }
  rotateZ(각도) { 지정한 각도만큼 z 로 회전 }
  perspective(길이) { 부모 요소에, 입체적으로 보일 수 있도록 깊잇값 지정 }

트랜지션

transition-property: 트랜지션의 대상을 지정
all | none | 속성이름(특정한 속성에만 적용할 경우)

transition-duration: 트랜지션을 실행할 시간을 지정
시간 - 예: 1s 2s;

transition-timing-function: 트랜지션의 실행 형태를 지정
트랜지션의 시작, 중간, 끝에서 속도를 지정해
전체 속도 곡선을 만들 수 있다
예
  linear: 천천히 - 점점 빨라짐 - 천천히 (기본값)
  ease: 속도 동일
  ease-in: 느리게 시작
  ease-out: 느리게 끝
  ease-in-out: 느리게 시작, 느리게 끝
  cubic-bezier(n, n, n, n): 베지에 함수 정의,
                           n 값은 0~1 사이만 가능

transition-delay: 트랜지션의 지연 시간을 지정
지연시간을 지정, 지정한 시간만큼 기다린 후 시작

transition: 위 4가지 속성을 한번에 지정
작성 순서는 상관없음
단, 시간 값이 두개(진행, 지연)이므로,
앞이 진행(transition-duration),
뒤를 지연(transition-delay) 으로 간주함

애니메이션

자바스크립트를 사용할 필요 없어짐

키프레임 설정
@keyframes 키프레임_이름 {
  from { 애니메이션 전 상태 }
  to { 애니메이션 후 상태 }
}

  animation-delay: 지연 시작 지정
  animation-direction: 애니메이션 종료 뒤 처음부터 시작할 지, 역방향으로 진행할 지 지정
    normal from 에서 to 로
    reverse to 에서 from 으로
    alternate normal 후 reverse
    alternate-reverse reverse 후 normal

  animation-duration: 애니메이션 실행 시간
  animation-iteration-count: 애니메이션의 반복 횟수 지정
  animation-name: @keyframes 로 설정해 놓은 중간 상태 지정
  animation-timing-function: 키프레임의 전환 형태 지정
  animation: 모든 속성을 한번에 묶어서 지정
